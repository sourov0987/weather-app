## Run Locally

Clone the project

```bash
  git clone https://github.com/the-sourov/weather-app.git
```

Go to the project directory

```bash
  cd "Weather App"
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm start
```
